# ogg_converter — пакетный конвертер OGG для Xiaozhi AI

Скрипт предназначен для массовой конвертации аудио в OGG‑формат, совместимый с устройствами Xiaozhi. Реализован на базе `ffmpeg-python`, поэтому **требуется** установленный `ffmpeg`.

Скачайте подходящую сборку с [официального сайта](https://ffmpeg.org/download.html) и добавьте `ffmpeg` в `PATH` или поместите исполняемый файл рядом со скриптом.

Поддерживаются конверсии между OGG и другими аудиоформатами, а также регулировка громкости.

## Создание и активация виртуального окружения

```bash
# создание окружения
python -m venv venv
# активация
source venv/bin/activate   # macOS/Linux
venv\Scripts\activate     # Windows
```

## Загрузка FFmpeg

Перейдите по ссылке выше, скачайте дистрибутив под свою ОС и:
- положите файл `ffmpeg` в каталог со скриптом **или**
- добавьте путь к нему в переменную окружения.

## Установка зависимостей

Выполняйте из активированного окружения:

```bash
pip install ffmpeg-python
```

## Запуск

```bash
python ogg_covertor.py
```
